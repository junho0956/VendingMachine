# Vending Machine
사용 스택 - html, css, javascript

### 목표
자바스크립트로 간단한 상태관리 연습하기

### 실행

> `$ git clone https://github.com/junho0956/VendingMachine.git`

> `$ cd ./server`

> `$ npm i`

> `$ npm start` (*default port : 3000*)

Open `index.html ` with Live server
